# cpp-png-binary
## Overview
PNG画像ファイルをバイナリから加工し、比較的高速に画像処理を行うことを目指しています。

## Feature
- 137×127の画像データ
  - 100回色反転
    - opencv: 1,116,891[μs]
    - 自作: 1,064,273[μs]
  - 100回リサイズ(横2.00倍、縦1.50倍)
    - opencv: 1,856,260[μs]
    - 自作: 1,614,714[μs]
- 3840×2879の画像データ
  - 100回色反転
    - opencv: 59,770,122[μs]
    - 自作: 122,924,651[μs]
  - 100回リサイズ(横2.00倍、縦1.50倍)
    - opencv: 97,884,240[μs]
    - 自作: 282,886,649[μs]
